# FOF-For-Africa-
A decentralized application (DApp) allowing users to burn Fairly Odd Fellas (FOF) NFTs based on their rarity (Uncommon, Elite, Epic, Legend) to receive an 80% ETH reward and trigger a 20% ETH donation to UNICEF Indonesia via the "FOF For Africa" fund. The burn also enables users to claim FOF-themed merchandise.


All financial contributions are exclusively processed through https://www.supportunicefindonesia.org/.
